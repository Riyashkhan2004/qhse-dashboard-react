# QHSE Dashboard

## About Project

QHSE Dashboard is a modern dashboard application for managing Quality, Health, Safety and Environment information.

## Technologies Used

- React
- TypeScript
- HTML
- CSS
- JavaScript

## Features

- Dashboard interface
- Data visualization
- Reports
- Responsive design
- Component based architecture

## Installation

Clone repository:
Install dependencies:


npm install


Run project:


npm run dev
